# HireSense


## Hour 1 – Project Setup & Understanding

### Project Name
**AI Resume & Mock Interviewer (HR-Tech)**

### What This Project Does
This project is an AI mentor for job seekers.

It:
- Explains **why a resume is getting rejected**
- Finds **missing or weak skills** compared to a job description
- Generates **mock interview questions** based on those weaknesses
- Gives feedback using the **STAR framework**

In short:  
**It tells users what is wrong with their resume and trains them to fix it.**
