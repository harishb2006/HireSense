# AI Resume & Mock Interviewer

An **AI career mentor** that explains *why* a resume gets rejected and then trains the user to fix those exact weaknesses.

---

## What It Does

1. Analyzes a resume against a specific job description
2. Shows missing or weak skills with clear explanations
3. Generates role-specific interview questions
4. Gives real-time feedback using the **STAR framework**

---

## User Flow

1. **Upload Resume (PDF)** + **Paste Job Description**
2. **AI Analysis** returns:

   * Match Score
   * Missing / weak skills
   * Section-wise feedback
3. **Dashboard** highlights gaps
4. Click **"Practice for this Role"**
5. **Mock Interview** with recruiter-style AI
6. Feedback + improvement suggestions

---

## Core Features

* **Semantic Skill Matching**
  Understands meaning, not just keywords

* **Targeted Mock Interviews**
  Questions based only on detected weaknesses

* **STAR-Based Feedback**
  Improves clarity, impact, and structure

---

## Optional Features

* **STAR Resume Rewriter** – rewrites weak bullets for impact
* **Job-Readiness Scorecard (PDF)** – resume + interview summary

---

## Why It’s Different

> Not a resume scanner.
> An AI mentor that tells you *why* you’re failing — and helps you fix it.
